    • opBinary

      Declaration

      pure nothrow Text opBinary(string op)(string text) if (op == "~" || op == "+");

      Concatenate the current string with another.

      Examples

      1. Font fnt = new Font("samples/font/arial.ttf", 12); Text t = new Text(font); t.setData("My new string"); t ~= "is great!"; // t draws now 'My new string is great' on screen.
      The example above is the same as if you do:
      1. t += "is great!";
      Both operators (~ and +) are allowed.
