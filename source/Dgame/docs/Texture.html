  • formatToBits

    Declaration

    pure nothrow @nogc ubyte formatToBits(Texture.Format fmt);

    Format a Texture.Format into the related bit count. If the format is not supported, it returns 0.

    Examples

    1. assert(formatToBits(Texture.Format.RGBA) == 32); assert(formatToBits(Texture.Format.RGB) == 24); assert(formatToBits(Texture.Format.BGRA) == 32); assert(formatToBits(Texture.Format.BGR) == 24);

  • bitsToFormat

    Declaration

    pure nothrow @nogc Texture.Format bitsToFormat(ubyte bits, bool reverse = false);

    Format a bit count into the related Texture.Format. If no bit count is supported, it returns Texture.Format.None.

    Examples

    1. assert(bitsToFormat(32) == Texture.Format.RGBA); assert(bitsToFormat(24) == Texture.Format.RGB); assert(bitsToFormat(32, true) == Texture.Format.BGRA); assert(bitsToFormat(24, true) == Texture.Format.BGR);

  • switchFormat

    Declaration

    pure nothrow @nogc Texture.Format switchFormat(Texture.Format fmt, bool alpha = false);

    Switch/Reverse Texture.Format.

    Examples

    1. assert(switchFormat(Texture.Format.RGB) == Texture.Format.BGR); assert(switchFormat(Texture.Format.RGB, true) == Texture.Format.BGRA); assert(switchFormat(Texture.Format.RGBA) == Texture.Format.BGRA); assert(switchFormat(Texture.Format.RGBA, true) == Texture.Format.BGRA);
